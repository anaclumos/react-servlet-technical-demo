# typescript-next-template

TypeScript Next Template

## Testing out

- `git clone` this repository.

```
git clone https://github.com/anaclumos/typescript-next-template
```

- `cd` into the directory.

```
cd typescript-next-template
```

- Install packages.

```
yarn
```

- Start Dev Mode

```
yarn dev
```
